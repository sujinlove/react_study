## 패키지 관리자 ( npm, yarn )

개발과정에서 필요한 패키지의 다운로드, 업데이트, 제거 등을 담당하는 명령어

## 설치하기

- npm install 패키지 이름 —save 
- 혹은
- yarn add 패키지 이름

## 삭제하기

- npm uninstall 패키지 이름 —save
- yarn remove 패키지 이름

## 리액트 프로젝트 생성하기

- npm create react-app 프로젝트 이름
- yarn create react-app 프로젝트 이름

## 생성된 프로젝트 실행하기

yarn start
